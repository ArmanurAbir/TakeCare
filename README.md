
# TakeCare – Healthcare Website

TakeCare is a responsive healthcare website built as part of a university group project. The website provides users with a clean and simple interface to access health-related information, interact with a basic health chatbot, and use a medication reminder feature.

## Project Overview

The aim of this project was to design and develop a user-friendly healthcare website using front-end web technologies. The website focuses on accessibility, clear navigation, responsive design, and interactive features that improve the user experience.

## My Role

I was responsible for integrating the final codebase, checking the functionality of all pages, improving interactivity, and supporting the overall front-end development of the project.

My contributions included:

- Integrating HTML, CSS, JavaScript, jQuery, and Bootstrap components
- Implementing and testing form validation
- Adding interactive features using JavaScript
- Supporting the health chatbot inside a Bootstrap modal
- Working on the medication reminder feature
- Checking page responsiveness and final functionality
- Fixing layout and usability issues before submission

## Features

- Responsive website layout
- Clean healthcare-focused user interface
- Health chatbot displayed in a Bootstrap modal
- Medication reminder feature with:
  - Medication name
  - Dosage
  - Time of consumption
- Form validation using JavaScript
- Smooth navigation between pages
- Interactive buttons and UI elements
- Accessible and user-friendly page structure

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap
- DOM Manipulation
- Form Validation
- Responsive Web Design

## Pages / Sections

The project includes healthcare-related pages and interactive components such as:

- Home page
- Health information sections
- Contact / form sections
- Health chatbot modal
- Medication reminder section

## What I Learned

Through this project, I improved my understanding of:

- Building responsive websites
- Structuring HTML pages clearly
- Styling pages using CSS and Bootstrap
- Using JavaScript for interactivity
- Validating forms on the client side
- Working with jQuery and Bootstrap components
- Integrating code from multiple team members
- Testing and improving final website functionality
- Collaborating in a group project environment

## Challenges Faced

Some challenges during the project included:

- Combining code from different group members into one working website
- Making the design consistent across pages
- Ensuring JavaScript features worked correctly
- Testing form validation and interactive components
- Improving responsiveness for different screen sizes

## Future Improvements

In the future, this project could be improved by adding:

- User login and account system
- Database storage for medication reminders
- More advanced chatbot responses
- Appointment booking system
- Backend support using Node.js, PHP, or Python
- Improved accessibility testing
- Dark mode option
